# Atlas 23.0.RC3 SDK 解压记录

本实验在 VMware Ubuntu 中下载并解压 Atlas 200I A2 / Ascend310B 对应的 HDK SDK 源码包。

## 下载包

Ascend-hdk-310b-sdk-soc_23.0.rc3.zip

## 解压后主要内容

外层解压后得到 Ascend310B-sdk 目录，其中包含：

- Ascend310B-source.tar.gz
- toolchain.tar.gz
- A500-A2-hdm_23.0.rc3.zip
- audio_sample.tar.gz
- hdmi_sample.tar.gz
- sdtool.tar.gz
- vrdtool.tar.gz
- efuse-tool.tar.gz
- emmc_burn_tool.tar.gz

## 下一步

继续解压 Ascend310B-source.tar.gz，并在源码目录中定位 Linux RTC 驱动文件。
